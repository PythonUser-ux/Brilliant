Factorial

Number Theory Level 2

n!n!n! is the factorial of nnn, where n!=1⋅2⋅3⋅...⋅(n−1)⋅nn!=1\cdot 2\cdot 3\cdot ...\cdot \left( n-1 \right) \cdot nn!=1⋅2⋅3⋅...⋅(n−1)⋅n. For example 4!=1⋅2⋅3⋅4=244!=1\cdot 2\cdot 3\cdot 4=244!=1⋅2⋅3⋅4=24.

If n!=215⋅36⋅53⋅72⋅11⋅13n!={ 2 }^{ 15 }\cdot { 3 }^{ 6 }\cdot { 5 }^{ 3 }\cdot { 7 }^{ 2 }\cdot 11\cdot 13n!=215⋅36⋅53⋅72⋅11⋅13 what is the value of nnn?

Well, there are two ways of solving this. One way is to rearrange the given value in the form of a factorial and keep count of the prime factors to get, n!=16!  ⟹  n=16n!=16! \implies \boxed{n=16}n!=16!⟹n=16​

Another way is to use reasoning. We see that there are 151515 2's in the expansion, so using the method of finding trailing zeros, we can also find that if there are 151515 2's in the expansion, then we have n∈{16,17}n\in \{16,17\}n∈{16,17} since if n<16n\lt 16n<16, there will be less number of 2's and if n>17n\gt 17n>17, there will be more number of 2's. Also, observe that 171717 is a prime number, so if n=17n=17n=17, there should be a 171717 in the given prime factorization. But there isn't! So, we have the result n=16n=16n=16 assuming the given factorial exists. We can verify the assumption easily.

Hence, the answer is n=16\boxed{n=16}n=16​ □_\square□​
