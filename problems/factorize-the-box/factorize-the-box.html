Factorize the Box

Geometry Level 2

A cuboid box has a height of xxx and a base diagonal (in blue) of length 2x2x2x.

Considering the triangle ABCABCABC separately, its height (in red) is yyy, partitioning the (blue) base into x−yx-yx−y and x+yx+yx+y, as shown above right.

If the volume of the box is 108, what is the value of yyy?


Hint: This wiki can help.


The answer is 3.

Relevant wiki: Sophie Germain Identity

The base area of the cuboid = ((x−y)2+y2)((x+y)2+y2)\sqrt{((x-y)^2 + y^2)((x+y)^2 + y^2)}((x−y)2+y2)((x+y)2+y2)​

By using Sophie Germain Identity, we can apply:

x4+4y4=((x−y)2+y2)((x+y)2+y2)x^4 + 4y^4 = ((x-y)^2 + y^2)((x+y)^2 + y^2)x4+4y4=((x−y)2+y2)((x+y)2+y2)

Hence, the base area = x4+4y4\sqrt{x^4 + 4y^4}x4+4y4​

Now since the triangle ABCABCABC is a right triangle, and let BDBDBD be the red height, x−yy=yx+y\dfrac{x-y}{y} = \dfrac{y}{x+y}yx−y​=x+yy​ because of similarity.

Then, x2−y2=y2x^2 - y^2 = y^2x2−y2=y2.

x2=2y2x^2 = 2y^2x2=2y2 or x=y2x = y\sqrt{2}x=y2​.

Therefore, the base area = x4+4y4=4y4+4y4=(22)y2\sqrt{x^4 + 4y^4} = \sqrt{4y^4 + 4y^4} = (2\sqrt{2})y^2x4+4y4​=4y4+4y4​=(22​)y2.

Then, the cuboid's volume = (22)y2x=4y3=108(2\sqrt{2})y^{2}x = 4y^3 = 108(22​)y2x=4y3=108.

As a result, y3=27y^3 = 27y3=27. y=3y = \boxed{3}y=3​.
